Tony's DH Projects


I am currently a graduate student at Vanderbilt University involved in a variety of projects related to Digital Humanities. This page strives to provide access to the code (via GitHub) and final products.

Syriaca.org Colophon Project

Description: Working in conjunction with Dr. David A. Michelson and Mr. Justin Arnwine, this project involves querying Manuscript data in the Syriac Reference Portal using XQuery to find all colophons and transform the XML data into HTML found on the Colophon Page

Chronicle of Edessa

Description: This project involves: 1) running a digitized version of the Chronicle of Edessa through Stanford CoreNLP software; 2) georeferencing all geographic places within the tokenized file with matched places found in the Syriac Gazetteer using XQuery, and; 3) transforming the georeferenced XML file into HTML using XQuery.
